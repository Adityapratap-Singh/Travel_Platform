# Travel App

This project has been restructured for separate frontend and backend deployment.

## Structure

- **frontend/**: React + Vite application
- **backend/**: Node.js + Express + MongoDB application

## Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs on http://localhost:5173 (or similar)

### Backend
```bash
cd backend
npm install
node index.js
```
Runs on http://localhost:3001
MongoDB must be running locally.

## Deployment

You can deploy the `frontend` folder to Vercel/Netlify/etc.
You can deploy the `backend` folder to Render/Heroku/Railway/etc.
